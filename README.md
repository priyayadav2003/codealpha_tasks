# CodeAlpha Data Analytics Internship - Task 01

## Project :Web Scraping

**Description :**
This project extracts quotes,authors,and tags from  [http://quotes.toscrape.com](http://quotes.tooscrape.com)using python and
BeautifulSoup.
The project scrapes multiple pages(first 3 pages) and stores the collected data in CSV file for analysis.

**scraper.py Script:**

- This python script contains all the code to scrape the website.
- Extracts **quotes**,**authors**,and **tags**.
- Prints first few quotes in the terminal for verification.

**CSV file (`quotations_data.csv`):**
- Automatically generated by `scraper.py`.
- Contains 3 columns:
   1. `quotations`:Text of the quotes
   2. `Authors_name`:author of each quote
   3. `Tags_list`:Tags associated with each quote

- You can open this file in **Excel,VSCode,or Google Sheets** to view the full dataset.

**Tools used:**
- Python
- Beautifulsoup
- Pandas
- Requests

**How to run :**
 1. Clone or download the repository.
 2. Make sure python  is installed along with required libraries:
 3. Install the required libraries:
     ```bash
     pip install requests beautifulsoup4 pandas
4. Open the project folder in VSCode.
5. Run the scraper script in terminal:
   ```bash
   python scraper.py
6. After running the script,the file 'quotations_data.csv' will be generated automatically.

Subject: "Sales Data Visualization Dashboard"

📌Project overview
  This project focuses on transforming raw sales data into meaningful
  visual insights using Python.
  The goal is to analyse sales performance by diiferent categories such as project type ,
  month and region through graphical representations.

  The project demonstrate how data visualization help businesses understand trends annd make better decisions.

🎯Objectives
  - Convert raw data into visual insights.
  - Analyse sales trends across months.
  - Identify top-performing products.
  - Understand sales distribution by region.
  - Build simple visualization dashboard using python.

🛠️Tools & Technologies
  - Python
  - Pandas: Data manipulation
  - Matplotlib: Basic plotting
  - Seaborn : Advanced data Visualization
  - VS Code: Development environment

📊Visualization included
   This project generates following graphs:
    1. Total sales by category
      - Bar chart showing which products generates the most revenue.
    2. Monthly sales Trend
      - Line chart displaying how sales change over different over months.
    3. Top products by sales
      - bar chart highlighting the best performing products.
    4. Sales Distribution by Region  
      - pie chart showing the percentage of sales from different regions.

All graphs are displayed together  in a dashboard layout.

📂Dataset
   A sample dataset is created within the python script containg the following columns:
    - Date
    - Product
    - Catagory
    - Region
    - sales 
    - Month
  This dataset simulates sales data for visualization practice.

▶️How to run  the project
  1. Install required libraries:
     pip install pandas
     matplotlib seaborn
  2. Run the python file:
     python DataVisualisation.py
  3. The visualization dashboard will appear on the screen.

📈Key Insights 
  1. Electronics category generates the highest sales.
  2. Some products consistently outperform others.
  3. Monthly trends helps understand peaks sales periods.
  4. Regional distribution highlights where most sales occur.

📌Project Overview
 This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand 
 the patterns, relationships,and insights releted to passenger survival .
 The analysis includes data inspection, missing value detection,statistical summaries, and visualizations.

⚙️Tools used 
 - Python 
 - Pandas
 - Matplotlib
 - seaborn

📂Dataset
  The dataset is used in this priject is Titanic Dataset, which contains information about passengers such as:
  - Passenger class
  - Name
  - Age 
  - Gender 
  - Fare
  - Survival status

Steps performed in EDA
  1. Importing Libraries
    -  Necessary Python libraries are imported for data analysis and visualization.
  2. Loading Dataset
    - The dataset is loaded using Pandas.
  3. Data Inspection
    - Displaying the first few rows of the dataset
    - Checking dataset structure and column information
  4. Missing Value Analysis
    - Identifying missing values in each column of the dataset.
  5. Statistical Summary
    - Using descriptive statistics to understand the distribution of numerical data.


📊 Data Visualization
   Several visualizations are created to understand patterns in the data:
     1. Survival Count Plot – Shows number of passengers who survived vs not survived.
     2.Survival by Gender – Compares survival rate between male and female passengers.
     3.Age Distribution – Displays distribution of passenger ages.
     4. Passenger Class Distribution – Shows number of passengers in each class.

📌Insights
    Some observations from the analysis include:
 * Female passengers had a higher survival rate compared to males.
 * Passenger class played a role in survival probability.
 * Most passengers were between the ages of 20–40.

▶️How to Run the Project
 1.Install required libraries:
    pip install pandas matplotlib seaborn
 2.Place the dataset file titanic.csv in the project folder.
 3.Run the Python script:
    python EDA_analysis.py


🎯Conclusion
   Exploratory Data Analysis helps in understanding the structure and important patterns in the Titanic dataset. The visualizations provide useful insights about survival trends among passengers.
